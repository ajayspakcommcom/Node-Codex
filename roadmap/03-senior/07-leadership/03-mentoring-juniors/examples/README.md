# Mentoring Examples

This folder contains practical artifacts for mentoring junior engineers in an enterprise engineering environment.

## Coverage

- mentoring checklist
- good vs bad mentoring behavior
- growth-plan example
- feedback examples
- onboarding and support plan
- incident mentoring guidance
- independent reasoning guidance

## Notes

- these examples are documentation-first because mentoring is primarily a capability-building and leadership topic
- the goal is to show how senior engineers help others grow without creating dependency
