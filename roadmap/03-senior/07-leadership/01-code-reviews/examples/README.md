# Code Review Examples

This folder contains practical review-oriented artifacts for enterprise code review work.

## Coverage

- review checklist
- blocking vs non-blocking feedback examples
- weak review vs strong review comment examples
- sample PR findings across correctness, maintainability, security, performance, and observability

## Notes

- these examples are intentionally documentation-first because code review is primarily a judgment and communication topic
- the goal is to show what strong senior review behavior looks like in practice
