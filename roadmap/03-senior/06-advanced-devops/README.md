# Advanced DevOps

## Purpose

This section focuses on the platform and rollout concerns behind large production systems.

At the senior level, advanced DevOps matters because teams need safe deployment strategies, repeatable infrastructure, and orchestration awareness that supports reliable system delivery.

## Topics

- Kubernetes fundamentals
- Helm
- blue-green and canary deployments
- Infrastructure as Code

## Enterprise Pointers

- understand the runtime platform your services depend on
- make deployment changes observable and reversible
- keep infrastructure reviewable and repeatable
- treat rollout strategy as part of reliability engineering
- avoid hiding operational complexity behind vague platform abstraction
