# Threat Modeling Code

This package models threat modeling as explicit assets, trust boundaries, risk scoring, and mitigation ownership rather than static diagrams with no engineering follow-through.

## Coverage

- asset and entry-point inventory
- trust boundary definition
- threat scenario scoring
- mitigation and residual-risk ownership
- review-ready threat summary generation
- maintainability-oriented threat model upkeep

## Notes

- this is an enterprise-style threat-modeling package with local validation support
- the goal is to keep threat review actionable for engineering teams, not only for security specialists
- files are intentionally structured to separate assets, boundaries, scenarios, and mitigation decisions
