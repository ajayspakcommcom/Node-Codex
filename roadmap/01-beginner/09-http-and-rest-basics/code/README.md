# HTTP And REST Enterprise TypeScript Examples

These examples focus on API contracts at the web boundary: methods, status codes, JSON, headers, cookies, resource design, validation, pagination, versioning, and maintainable response patterns.

## Files

- `01-request-lifecycle-and-methods.ts`
- `02-status-codes-and-response-semantics.ts`
- `03-json-headers-and-cookies.ts`
- `04-restful-resource-design.ts`
- `05-boundary-validation.ts`
- `06-consistent-response-shapes.ts`
- `07-pagination-filtering-and-sorting.ts`
- `08-api-versioning-and-auth-headers.ts`
- `09-common-api-mistakes.ts`
- `10-interview-questions.ts`
- `11-practice-exercises.ts`

## Shared Support

- `shared/http-types.ts`
- `shared/response-helpers.ts`
- `shared/validation.ts`

## Run

Use a TypeScript runner such as `tsx`:

```bash
npx tsx roadmap/01-beginner/09-http-and-rest-basics/code/01-request-lifecycle-and-methods.ts
```
