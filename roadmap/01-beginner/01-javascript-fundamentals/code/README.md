# JavaScript Fundamentals Code Examples

These examples are written for backend-oriented Node.js learning with an enterprise mindset.
They now use TypeScript files and are intended to follow strict TypeScript conventions rather than a plain extension rename.

- `07-equality-and-nullish-handling.ts`

## Run

Use a TypeScript runner such as `tsx`:

```bash
npx tsx roadmap/01-beginner/01-javascript-fundamentals/code/01-variables-and-scope.ts
```

Replace the filename with any example you want to run.

## Type Checking

This topic also includes a local `tsconfig.json` so the examples can be type-checked with strict settings when TypeScript is available.
