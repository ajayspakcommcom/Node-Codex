# Beginner Projects For Enterprise Node.js And TypeScript

## Purpose

This folder contains the practical project work for the `01-beginner` section.

These projects are not toy demos. They are meant to combine the beginner topics into realistic backend implementations with enterprise structure, clear boundaries, and maintainable code organization.

## Projects

- `01-rest-api-auth-crud`
- `02-simple-file-upload-service`

## Enterprise Goal

Each project should reinforce:

- clean API structure
- validation at the boundary
- service and repository separation
- safe error handling
- configuration discipline
- basic security awareness
- testable and maintainable structure

## Why This Matters

Topic-by-topic learning is necessary, but projects are where the concepts start working together.

These two beginner projects are the bridge between:

- isolated topic understanding
- real backend application building

## Expected Outcome

After completing these projects, you should be able to:

- build a structured backend service from scratch
- apply beginner Node.js concepts in one codebase
- organize routes, middleware, services, and persistence correctly
- prepare for more advanced architecture in the `02-intermediate` section
